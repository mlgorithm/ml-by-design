\chapter*{Part Synthesis: Foundations}
\addcontentsline{toc}{chapter}{Part Synthesis: Foundations}

\begin{whychaptermatters}
This part built the book's first complete workflow. Students now have a way to frame a decision, choose what good behavior means, protect the claim with held-out evidence, test a serious baseline, and diagnose what structure is still missing when that baseline fails.
\end{whychaptermatters}

\begin{chaptersummary}
\item \textbf{Question this part taught.} What problem are we solving, what counts as good behavior, what evidence protects the claim, and what missing structure justifies the next model family?
\item \textbf{Artifact chain this part added.} Framing memo $\rightarrow$ metric-and-action worksheet $\rightarrow$ evaluation plan $\rightarrow$ first baseline design sheet $\rightarrow$ structure diagnosis sheet.
\item \textbf{What a student can now do.} Turn a vague ML request into a defended baseline workflow and justify the next model family from failure type instead of from fashion.
\end{chaptersummary}
