# Third-Party Materials

The blanket licenses in the repository root and in `companion/` do not apply to
third-party materials unless a file explicitly says otherwise.

At the moment, the main third-party assets are:

- `figures/external/road_scene_crossing_small.jpg`
- `figures/external/syntax_tree_clean.png`
- `figures/external/zech2018_g003_small.png`

These assets should be treated as externally sourced editorial materials and
not as original repository content for relicensing purposes.

Before a public print release or Zenodo deposit, verify that each third-party
asset has one of the following:

- permission to redistribute
- a license compatible with the intended release
- a clear replacement plan if redistribution rights are uncertain
