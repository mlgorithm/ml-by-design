Release workflow

What to do when you want a DOI

  1. Enable this repository in Zenodo.
  2. Create a tagged GitHub release for the edition you want to cite.
  3. Wait for Zenodo to archive the release and mint the DOI.
  4. Update this site and the README with the minted DOI.
